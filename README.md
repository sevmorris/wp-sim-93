# Buckingham Sim 93

A browser-based terminal simulator and text adventure loosely based on a real place and time.

Philadelphia. West Philly. October 1993. You've just woken up in your apartment. It's raining. Nothing on the agenda.

There's no score. No inventory puzzle to solve. No boss fight. Just a small living room, some records, a cat named Cracker, and whatever you feel like doing with your morning.

**[Play it here](https://sevmorris.github.io/buckingham-sim-93/)**

---

Built as a single self-contained HTML file — no dependencies, no build step. Type `help` in the game for a list of commands, or just start poking around.
